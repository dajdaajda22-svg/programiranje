Interaktivna prezentacija o osnovama programiranja

Šta je programiranje?

Definicija:

Programiranje je proces pisanja instrukcija (koda) koje računar izvršava kako bi riješio određeni problem ili obavio zadatak.

Programski jezik:

  • Način komuniciranja sa računarom
  • Skup pravila i sintakse
  • Prevodi ljudske naredbe u mašinski kod

Zašto učiti programiranje?

✅ Logičko razmišljanje - Razvoj analitičkih vještina
✅ Rješavanje problema - Kreativne solucije
✅ Karijerne mogućnosti - Visoke plate
✅ Kreativnost - Pravljenje aplikacija

QBasic - Početak programiranja

QBasic (Quick Beginners All-purpose Symbolic Instruction Code)

Karakteristike:

  • Jezik za početnike (1991. godina)
  • Jednostavna sintaksa
  • Idealan za učenje osnova
  • Integriran editor i interpreter

Primjer - Pozdrav:

CLS PRINT "Zdravo, dobrodošli u QBasic!" PRINT "Moje prvo programiranje" INPUT "Unesite vaše ime: ", ime$ PRINT "Drago mi je, "; ime$ END

QBasic - Osnovne naredbe

Važne naredbe:

CLS ' Čisti ekran PRINT "tekst" ' Ispisuje tekst INPUT "pitanje", var ' Unos podataka LET a = 5 ' Dodjeljivanje vrijednosti REM komentar ' Komentar ' Aritmetičke operacije a = 10 + 5 ' Sabiranje b = 10 - 5 ' Oduzimanje c = 10 * 5 ' Množenje d = 10 / 5 ' Djeljenje

Primjer - Kalkulator:

INPUT "Unesite prvi broj: ", broj1 INPUT "Unesite drugi broj: ", broj2 zbir = broj1 + broj2 PRINT "Zbir je: "; zbir

QBasic - Grananje i petlje

IF naredba (Grananje):

INPUT "Unesite vašu ocjenu: ", ocjena IF ocjena >= 5 THEN PRINT "Položili ste!" ELSE PRINT "Niste položili." END IF

FOR petlja (Ponavljanje):

' Ispisuje brojeve od 1 do 10 FOR i = 1 TO 10 PRINT i NEXT i ' Tablica množenja INPUT "Unesite broj: ", broj FOR i = 1 TO 10 PRINT broj; " x "; i; " = "; broj * i NEXT i

C++ - Moćan programski jezik

C++ (1985. godina - Bjarne Stroustrup)

Karakteristike:

  • Objektno-orijentisani jezik
  • Brz i efikasan
  • Koristi se za: igre, operativne sisteme, aplikacije
  • Osnova za mnoge moderne jezike

Struktura C++ programa:

#include <iostream> using namespace std; int main() { cout << "Zdravo iz C++!" << endl; return 0; }

Objašnjenje:

  • #include - uključuje biblioteke
  • main() - glavna funkcija
  • cout - ispis na ekran
  • return 0 - kraj programa

C++ - Varijable i tipovi podataka

Tipovi podataka:

#include <iostream> using namespace std; int main() { // Cijeli brojevi int godine = 25; // Decimalni brojevi float cijena = 15.99; double visina = 175.5; // Slova char slovo = 'A'; // Tekst string ime = "Kenan"; // Logičke vrijednosti bool tacno = true; // Ispis cout << "Ime: " << ime << endl; cout << "Godine: " << godine << endl; cout << "Cijena: " << cijena << " KM" << endl; return 0; }

C++ - Operatori i izračunavanja

Aritmetički operatori:

#include <iostream> using namespace std; int main() { int a = 10, b = 3; cout << "Sabiranje: " << (a + b) << endl; // 13 cout << "Oduzimanje: " << (a - b) << endl; // 7 cout << "Mnozenje: " << (a * b) << endl; // 30 cout << "Djeljenje: " << (a / b) << endl; // 3 cout << "Ostatak: " << (a % b) << endl; // 1 return 0; }

Relacioni operatori:

int x = 5, y = 10; x == y // Jednako (false) x != y // Različito (true) x > y // Veće (false) x < y // Manje (true) x >= y // Veće ili jednako (false) x <= y // Manje ili jednako (true)

C++ - Grananje i petlje

IF-ELSE:

int ocjena; cout << "Unesite ocjenu: "; cin >> ocjena; if (ocjena >= 5) { cout << "Cestitamo, polozili ste!" << endl; } else if (ocjena >= 3) { cout << "Slaba ocjena, ali prolaz." << endl; } else { cout << "Niste polozili." << endl; }

FOR petlja:

// Brojevi od 1 do 5 for (int i = 1; i <= 5; i++) { cout << i << " "; } // Tablica množenja int broj = 7; for (int i = 1; i <= 10; i++) { cout << broj << " x " << i << " = " << (broj*i) << endl; }

C++ projekat u Visual Studio

Kreiranje projekta:

  1. File → New → Project
  2. Izaberi "Console App" (C++)
  3. Daj ime projektu
  4. Click "Create"

Primjer - Kalkulator:

#include <iostream> using namespace std; int main() { double broj1, broj2; char operacija; cout << "=== KALKULATOR ===" << endl; cout << "Unesite prvi broj: "; cin >> broj1; cout << "Unesite operaciju (+, -, *, /): "; cin >> operacija; cout << "Unesite drugi broj: "; cin >> broj2; switch(operacija) { case '+': cout << "Rezultat: " << (broj1 + broj2) << endl; break; case '-': cout << "Rezultat: " << (broj1 - broj2) << endl; break; case '*': cout << "Rezultat: " << (broj1 * broj2) << endl; break; case '/': if(broj2 != 0) cout << "Rezultat: " << (broj1 / broj2) << endl; else cout << "Greska: Djeljenje sa nulom!" << endl; break; default: cout << "Nepoznata operacija!" << endl; } return 0; }
